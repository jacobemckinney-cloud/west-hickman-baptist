# West Hickman Baptist Church Website

This project is a multi-page static church website for **West Hickman Baptist Church** in Hickman, Kentucky. It is built with plain HTML, CSS, and JavaScript so it can be hosted on GitHub Pages and easily edited later.

## File Structure

- `index.html` - Home page
- `about.html` - Church mission, history, and leadership
- `ministries.html` - Ministry overview and highlights
- `visit.html` - Plan your visit details and expectations
- `contact.html` - Contact details, map link, and full service times
- `styles.css` - Shared site styling
- `script.js` - Mobile navigation behavior
- `assets/images/` - Local folder for church photos

## How to Edit Text

1. Open the page you want to update (`index.html`, `about.html`, etc.).
2. Edit headings, paragraphs, or schedule text.
3. Save and preview in a browser.

## How to Replace Images

1. Add new image files to `assets/images/`.
2. Update the `src` paths in the HTML where placeholders are used.
3. Keep meaningful `alt` text for accessibility.
4. Search for comments like `Replace this image with a real church photo` for quick replacement spots.

## Publish with GitHub Pages

1. Push the site to GitHub.
2. In the repository settings, open **Pages**.
3. Set source to the `main` branch (root).
4. Save, then wait for the site URL to be generated.
